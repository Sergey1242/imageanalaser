# Классификатор изображений с помощью нейросети ResNet50

## Что делает этот код?
Этот код загружает все фотографии из папки images, анализирует каждую с помощью готовой нейросети ResNet50 (которая обучена распознавать 1000 разных объектов, включая животных, предметы, технику) и определяет, что именно изображено на фото. Результат сохраняется в файлы Excel (CSV) прямо в папку с программой.

## Как это использовать? Пошаговая инструкция:

## Шаг 1. Скачать проект
На странице репозитория нажмите зеленую кнопку Code, выберите Download ZIP, распакуйте архив в любую удобную папку на компьютере.

## Шаг 2. Открыть VSC/PYCHARM/CMD
Разархивируйте файлы в папку в любое удобное место, потом запустите VSC/PYCHARM и нажмите FILE - Open folder и выберите тут папку в которую вы разархивировали файлы
Либо
Нажмите Win+R, введите cmd и нажмите Enter. Теперь с помощью команды cd перейдите в папку с проектом. Например: cd C:\Users\ВашеИмя\Downloads\Practice3-master

## Шаг 3. Установить нужные библиотеки
В командной строке по очереди введите две команды. После каждой нажимайте Enter и ждите окончания установки:

pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu

pip install pillow pandas

## Шаг 4. Добавить свои фотографии
Найдите в папке с проектом папку images. Положите в неё все свои фотографии, которые хотите распознать. Поддерживаются форматы .jpg, .jpeg, .png. Не кладите больше 100-200 фото за раз могут быть зависания.

## Шаг 5. Запустить программу
Запустите файл main.py

Программа начнет работать. Вы увидите в командной строке, как она обрабатывает каждое фото. Подождите, пока надпись "Готово" не появится. Обычно занимает 1-2 минуты на 100 фото.

## Шаг 6. Посмотреть результаты
После завершения в папке с проектом появятся три файла:

results.csv - полная таблица для всех фото. Там указано, что распознано на каждом фото и насколько нейросеть уверена в ответе (0.0 - совсем не уверена, 1.0 - абсолютно уверена).

successful_predictions.csv - только те фото, где нейросеть была уверена (уверенность больше 0.30).

uncertain_predictions.csv - фото, которые нейросеть распознала с низкой уверенностью.

## Как открыть эти файлы? Обычным Excel или любой программой для таблиц. Просто кликните два раза по файлу.

## Что означают цифры в таблице?
Уверенность: число от 0 до 1. Чем ближе к 1, тем правильнее ответ. Например, 0.95 - нейросеть почти не сомневается. 0.21 - нейросеть скорее не уверена.

Класс 1, Класс 2, Класс 3 - 5 вариантов того, что изображено на фото, от самого вероятного к наименее вероятному.

Вероятность 1, Вероятность 2 - насколько вероятен каждый из этих вариантов.

## Статус: "Удачное распознавание" если уверенность >=0.30 или "Сомнительное распознавание" если уверенность меньше.

## Если что-то пошло не так:

Ошибка "No module named torch" - вы забыли установить библиотеки. Вернитесь к шагу 3.

Ошибка "FileNotFoundError: images" - вы запускаете программу не из той папки. Проверьте, что командная строка открыта именно в папке, где лежит файл main.py и папка images.

Ошибка "PIL.UnidentifiedImageError" - в папке images лежит не картинка (например, текстовый файл или поврежденное фото). Удалите оттуда всё, кроме фотографий.

Файлы CSV не открываются в Excel - они откроются, если кликнуть правой кнопкой мыши -> "Открыть с помощью" -> выбрать Excel или любой браузер.

Если нейросеть неправильно распознает объекты - это нормально. ResNet50 обучена только на 1000 категориях. Она знает 120 пород собак, но не знает, что такое "Айфон" или "Самсунг". Но она может назвать машину "гоночным автомобилем", а кошку - "египетской кошкой". Чем проще и четче фото, тем лучше результат.

## Как изменить порог уверенности?
Если программа слишком часто пишет "Сомнительное распознавание", откройте файл main.py в блокноте, найдите 20 строчку где - CONFIDENCE_THRESHOLD = 0.30 и замените 0.30 на 0.20 (менее строгий порог) или 0.40 (более строгий). Сохраните файл и запустите программу заново.
